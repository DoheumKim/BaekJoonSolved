### 문제 설명

<p>백준 온라인 저지에는 "두 트리"라는 이름의 문제가 4개 있다. 대회가 너무 쉬울 것 같다는 검수진의 항의를 받은 한 출제자는 가장 어려운 두 트리를 더 어렵게 만들기로 했다.</p>
<p>$N$개의 정점과 서로 다른 $2(N-1)$개의 간선으로 이루어진 그래프가 주어진다. 각 정점은 1번부터 $N$번까지, 각 간선은 1번부터 $2(N-1)$번까지 번호가 부여되어 있다. $i$번 간선은 $A_i$번 정점과 $B_i$번 정점을 서로 연결한다. ($1 \le i \le 2(N-1)$)</p>
<p>각 간선을 빨강 혹은 파랑으로 칠하자. 빨간 간선으로 이루어진 그래프와 파란 간선으로 이루어진 그래프가 각각 트리가 되도록 간선을 칠할 수 있는지 판별하라.</p>
