### 문제 설명

<p>$N$개의 정점과 $M$개의 간선으로 이루어진 가중치 있는 무방향 그래프가 주어진다. 정점은 1번부터 $N$번까지 번호가 붙어 있고, 간선은 1번부터 $M$번까지 번호가 붙어 있다.</p>
<p>그래프의 최소 스패닝 트리(MST)는 다음과 같이 정의된다.</p>
<ul>
<li>주어진 그래프의 모든 정점을 연결하는 부분 그래프 중에서 간선 가중치의 합이 최소인 트리</li>
</ul>
<p>그래프의 최소 병목 스패닝 트리(MBST)는 다음과 같이 정의된다.</p>
<ul>
<li>주어진 그래프의 모든 정점을 연결하는 부분 그래프 중에서 간선 가중치의 최댓값이 최소인 트리</li>
</ul>
<p>입력으로 주어진 그래프에 대해서, MST이면서 MBST가 아닌 것과, MBST이면서 MST가 아닌 것을 각각 찾아라.</p>
